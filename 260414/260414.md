# command substitution

command substitution bedeutet, dass man den output eines Kommandos als Variable speichern kann.



# Anzahl Einträge in einem Verzeichnis

```sh
n=$(ls "$1"| wc -l)
echo "Es sind $n Einträge vorhanden"
```
ls | wc -l zählt die Einträge im aktuellen Verzeichnis. "$1" schaut für ls im übergebenen Verzeichnis nach, allerdings darf man sich nicht im übergeben Verzeichnis befinden (einfach zurück mit cd..). Diesen output habe ich in eine Variable gespeichet und später ausgegeben.

Unterschied zwischen : 
```
ls
ls | cat
echo "$(ls)"
```
- ls gibt Dateien direkt in der Konsole aus
- ls | cat leitet die Ausgabe durch einen Pipe weiter, weil ls in einen strom schreibt und cat ihn wiedergibt.
- echo  "$(ls)" führt ls aus und setzt das Ergebnis als Text in echo ein, alles wird beim substituieren zum String

# Tage bis zum Ball

```sh
ball_datum="2027-01-16"

# aktuelles datum in sekunden seit epoch
heute=$(date +%s)

# ball datum in sekunden seit epoch
ball=$(date -d "$ball_datum" +%s)

# differenz in sekunden umrechnen
diff=$((ball - heute))

# sekunden in tage umrechnen
tage=$((diff / 86400))

echo "Es sind noch $tage Tage bis zum HTL Ball ($ball_datum)"
```
Zuerst rechne ich die Sekunden seit Epoch aus (1970-01-01 00:00 UTC). Danach die Sekunden vom Ball Datum, und danach dividiere ich die Differenz durch die Anzahl Sekunden für einen Tag (86400).

# Zufälliger Satz

```sh
#!/bin/bash

file="wordlist.txt"

anzahl=$(cat $file | wc -l)

for i in {1..5}
do
# Zeilennummer
    random=$((RANDOM % anzahl + 1))

    word=$(sed -n "${random}p" "$file") 

    satz="$satz $word"
done

echo $satz
```
Zuerst habe ich mit file den Dateipfad definiert. Danach mit wc -l die Anzahl der Zeilen ermittelt und durch die Pipe die wordlist als input benutzt. In der for-Schleife generiere ich eine Zeilennummer mit dem üblichen Random Syntax (RANDOM modulo range). Hier wird dann auch die ermtittelte random-Zahl in die random Variable substituiert. Dann wird die Zeile ausgewählt aus der Datei und in eine word Variable gespeichert. Da "$random p oder random_p" nicht geht, braucht man curly braces ({random}p). Zuletzt wird alles in die satz Variable gespeichert. Am Anfang ist "satz" noch leer, deshalb wird nur "wort" hinzugefügt. In die nächsten Durchläufe wird dann der Satz gebildet (quasi wie +=).
